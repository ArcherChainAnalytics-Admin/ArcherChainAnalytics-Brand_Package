# ACA Brand Governance Standards

## 1. Logo Implementation
- **Source:** Use `assets/branding/logo_primary.pdf` for all web production to ensure vector transparency.
- **Clear Space:** Maintain a minimum margin of 20% of the logo's diameter on all sides.
- **Backgrounds:** The logo must only appear on Obsidian (#000000) or Clarity White (#FFFFFF) backgrounds.
