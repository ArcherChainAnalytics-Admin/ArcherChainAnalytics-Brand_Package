2. Brand Identity Overview
Archer Chain Analytics operates with a visual identity built on precision, sovereignty, and technical clarity.
The brand system is designed for:

Modular product families

High‑contrast, institutional interfaces

Long‑term auditability and consistency

Cross‑platform deployment (web, mobile, print)

3. Logo Governance
3.1 Primary Mark
The official Archer Chain Analytics logo is a circular emblem featuring:

The archer figure

The dual‑ring “ARCHER CHAIN ANALYTICS” perimeter

The central symbolic elements representing analytics, sovereignty, and technological focus

3.2 Usage Rules
Do not alter proportions, colors, or geometry.

Do not add effects, shadows, or outlines.

Maintain clear space equal to the height of the inner ring.

Use only approved color variants (black, gold, white).

Raster versions are for screen; vector versions are for print and scaling.

4. Color Standards
Purpose	Color	Hex
Primary Black	Black	#000000
Primary Gold	Gold	#D4AF37
Primary White	White	#FFFFFF

These values form the non‑negotiable palette for all brand surfaces.

5. Typography
Primary Typeface
Inter (UI, web, product surfaces)

Secondary Typeface
Montserrat (headings, print, large‑format materials)

Typography must remain clean, geometric, and high‑contrast.
